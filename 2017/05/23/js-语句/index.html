if,switch,while,continue,break,for

Posted on 2017-05-23 | In js

if

太常用太简单,不说了

switch

原理: n的值会与每个case的值作比较,如果匹配,会执行相应的代码块,否则不会执行,往下继续运行
每句结束必须要break

while 和 do while

语法: while(条件){ 循环体 }
do { 循环体 } while (条件)
do while 和 while 不同的是. 他会保证让循环体至少会执行一次, 因为他是先执行代码然后在判断条件

for

太常用太简单,不说了

break和continue

break是彻底结束当前循环,直接执行循环体后面的代码
continue是逃过本次循环,然后整个循环体会继续执行
